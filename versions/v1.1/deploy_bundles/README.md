# Deploy Bundles

Deploy bundles are review artifacts that bind a SkillsBench source revision,
runtime image digests, a task-set manifest digest, and the full prebuilt task
environment image map used by the AgentBeats worker.

Use `skillsbench-v1.1.json` as the operator input for the `skillsbench-v1.1`
AgentBeats registration and public-readiness evidence assembly.
